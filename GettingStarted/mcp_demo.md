## MCP Demo
To use the MCP server with DAB, the version needs to be 1.7+, and v2.0+ is recommended for bug fixes.

### Add the server
In VS Code, command palette, MCP: Add Server
- enter the URL: http://localhost:5000/mcp
- MCP:List Servers, start

This should see 7 tools (or more) listed in the output terminal

CTRL+ALT+I should open the copilot chat
